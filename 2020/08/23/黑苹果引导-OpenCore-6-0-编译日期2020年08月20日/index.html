黑苹果引导-OpenCore 6.0 编译日期2020年08月20日

置顶 | 发表于 2020-08-23 更新于 2020-08-25 分类于 opencore 阅读次数： Changyan：

作者：www.github.com/acidanthera/OpenCorePkg
具有开发SDK的OpenCore引导程序。

说明：

本文章提供的下载文件为本人自编译版本如果出现任何问题请前往上方官方GIT仓库下载正式版。

更新说明：

“V6.1 2020-08-14”
添加文本渲染器的builtinext变量以支持老式笔记本电脑
修正RSDP ACPI表校验和重新计算
添加对10.13+immutablekernel加载的支持
修复在11.0 kext注入中解决一些符号为零的问题
限制引导管理访问以减少OpenCanopy的大小

V6.0 2020-08-11”
增加了iMac20,1机型
修复了老旧Xeon cpu显示名称，如Xeon E5450
添加了kext注入支持，无需内核缓存
添加Comet Lake-LP HDA设备代码
修正了SATA控制器上的操作系统引导选择问题
优化代码

“V6.0 2020-08-03”
更新SSDT-EC-USBX.dsl
OcCpuLib：修复了i9 CPU（如7920X）的显示名称
优化代码
优化代码
更新了SMBIOS和其他设备的内置固件
增加了MacBookPro16,4模版
例行更新
优化部分代码
修正了DxeIpl中ACPI复位寄存器检测的问题
在KernelCollection中注入了最新预览版本（macOS Big Sur ）代码，进测试非10代的机器，大多数可以正常升级安装并顺利的进入到macOS Big Sur ，全新安装在测试中。

v0.6.0
修复使用 AudioDxe 导致声音损坏
修复 Opencanopy 中 Fw 图标选择
修复 APFS 驱动器加载
添加 Comet Lake U HDA 设备代码
修复非 intel 平台上的音频位置
添加 FirmwareResetSystem 到首选模式
替换 NVRAM 变量 BlacklistAppleUpdaterun-efi-updater
重置ACPI 里 FadtEnableReset
修复使用 PXE 扩展启动选项引起的死机
更新 EDK II 包 edk2-stable202005
增加 quirk 选项以提高笔记本的稳定性
修复了 0 张幻灯片不可用时平台上的幻灯片选择
修复了由DEBUG版本未匹配路径导致的问题
将utility重命名为 consistencyConfigValidityocvalidate
添加 APFS 用于解决较旧的固件问题
添加 quirkAvoidRuntimeDefrag 支持 macOS11.0 Booter
修复了 DP1 的 macOS quirk
更新无 NVRAM 的导致的引导选择脚本错误
添加 ProvideConsoleGop 的UGA兼容性
添加 GOPUgaPassThroug 支持 Uga 协议
添加了协议实现和重写AppleFramebufferInfo
禁用文件日志记录时修复了串行初始化
修复了关于 Meron 和类似 CPU 的 FSB 频率报告
修复了 OpenCanopy 中不正确的卷图标尺寸要求
v0.5.9
在OpenCanopy中添加了完整的HiDPI支持
通过使用CoreText改进了OpenCanopy字体渲染
固定灯光和自定义背景字体渲染
Boot####在启动项列表中增加了对选项的支持
HideSelf通过模式识别删除BOOTx64.efi
添加BlacklistAppleUpdate以避免Apple FW更新
修复了意外工具和NVRAM重置引导的默认情况
修复了无法识别的选择com.apple.recovery.boot条目
更改了NVRAM重置，不清除BootProtect启动选项
禁用选择器UI时提高了引导性能
外部故障时强制使用内置选择器
修复了有关空NVRAM变量的警告（例如rtc-blacklist）
已添加ApplePanic以在ESP根目录上存储紧急日志
修复ReconnectOnResChange即使不更改分辨率也可以重新连接的问题
修复了OpenCanopy显示外部驱动器的内部图标的问题
修复了OpenCanopy启动Shell并在其上显示文本的问题
向OpenCanopy添加了部分热键支持（例如Ctrl + Enter）
添加了内置文本渲染器与Shell页面模式的兼容性
固定FadtEnableReset过小的FACP桌子和一些笔记本电脑
使用QEMU 5.0和KVM加速器修复了CPU检测崩溃
RequestBootVarFallback由于大量错误已删除
添加了DeduplicateBootOrderUEFI怪癖
DirectGopCacheMode因无效而删除
修复了日志耗尽导致启动失败的断言
修复了内置文本渲染器无法提供ConsoleControl的问题
固定兼容性仅兼容blit的GOP（例如OVMF Bochs）
修复了#在DeviceProperty和NVRAM中忽略的问题Delete
更名Block到Delete在ACPI，DeviceProperties和NVRAM
添加了MacBookPro16,2和MacBookPro16,3型号代码
添加了PCI设备扫描策略支持（例如VIRTIO）
改进了AudioDxe中的播放性能
更新了SMBIOS及其它内置固件的内置版本
为新的CPU类型增加了改进的CPU类型检测
添加了ConfigValidity实用程序并改进了配置验证
添加了用于串行调试日志记录的串行端口初始化
禁用空调试日志文件创建以避免ESP混乱
添加TscSyncTimeout了解决调试内核断言的怪癖
为祝福模型添加了一流的Windows支持
修复LapicKernelPanic了10.9上的内核怪癖
添加了CrScreenshotDxe驱动程序的预构建版本
固定的Hyper-V频率检测兼容性
SysReport为DEBUG构建添加了选项以转储系统信息
修复了执行键盘输入时某些AMD固件崩溃的问题
